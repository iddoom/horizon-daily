## [Strix：用于自动发现和修复漏洞的开源 AI Agent 工具](https://github.com/usestrix/strix) ⭐️ 6.0/10

Strix 是一款开源的 Python 工具，目前在 GitHub 上备受关注。它利用自主 AI Agent 动态发现、验证并修复软件应用程序中的漏洞。该项目近期的参与度激增，在过去 24 小时内获得了 41 个 star 和 3 个 fork。 该项目代表了大型语言模型（LLM）在 DevSecOps 领域的实际应用，能够将通常需要大量人类专业知识的复杂安全任务自动化。通过将这项技术开源，Strix 让开发团队更容易接触到高级的漏洞修复技术，从而帮助他们将安全性无缝集成到持续交付流程中。 Strix 充当自主 AI Agent，能够执行代码、探索应用程序，并使用概念验证来验证已识别的漏洞。尽管该工具在自动化安全工作流程方面展现出潜力，但其目前的 GitHub 参与度指标表明，它仍处于被更广泛的开发者社区采用和验证的早期阶段。

ossinsight · usestrix · 7月4日 08:15

**背景**: DevSecOps 是将安全实践集成到 DevOps 流程中的一种方法，旨在自动化安全测试，在不牺牲安全性的前提下实现快速、持续的软件交付。最近，业界一直在探索使用由大型语言模型驱动的自主程序——即 AI Agent——来彻底改变漏洞研究。谷歌的 Big Sleep AI Agent 等知名案例展示了如何利用人工智能在被恶意利用之前，主动发现复杂的安全漏洞。
