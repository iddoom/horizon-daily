## [欧盟议会通过"聊天控制 1.0"大规模监控扩展法案](https://www.patrick-breyer.de/en/eu-parliament-greenlights-chat-control-1-0-breyer-our-children-lose-out/) ⭐️ 9.0/10

欧盟议会已批准延长"聊天控制 1.0"法案，允许美国科技公司在没有事先怀疑的情况下大规模扫描私人消息和电子邮件，直至 2028 年。尽管实际投票的多数议员（314 票反对、276 票赞成、17 票弃权）反对该法规，但否决动议未能获得全体议员中 361 票的绝对多数要求。 这一决定对整个欧盟的数字隐私、端到端加密和公民自由具有巨大影响，因为它以保护儿童为名将私人通信的大规模监控常态化。它还为如何利用议会程序技巧通过缺乏真正民主支持的立法开创了危险先例。 该法规影响 Instagram、Discord、Snapchat、Skype 和 Xbox 等平台上的直接消息，以及通过 Gmail 和 Apple iCloud 发送的电子邮件。公共社交媒体帖子和云存储文件在没有这项法律的情况下已经可以被扫描，而投票被策略性地安排在暑假前的最后一次欧洲议会会议上，有 113 名议员缺席。

hackernews · rapnie · 7月9日 11:03 · [社区讨论](https://news.ycombinator.com/item?id=48843923)

**背景**: 聊天控制是指欧盟的《防止和打击儿童性虐待条例》（CSAR），最初由专员 Ylva Johansson 于 2022 年 5 月提出。该法规允许扫描私人通信以查找 CSAM 材料，但隐私倡导者认为它通过客户端扫描（CSS）破坏了端到端加密，即在加密前或解密后扫描消息内容。欧盟此前在 2024 年将自愿性的聊天控制 1.0 法规延长至 2026 年 4 月，而这次最新投票进一步扩展了大规模扫描能力。

## [Ollama 完成 6500 万美元 B 轮融资，用户量近 900 万](https://techcrunch.com/2026/07/09/popular-open-source-ai-developer-tool-ollama-raises-65m-grows-to-nearly-9m-users/) ⭐️ 7.0/10

Ollama 在由 Theory Ventures 领投的 B 轮融资中筹集了 6500 万美元，此前该公司已完成了 1500 万美元的 A 轮融资。该公司还报告称，其用户群已增长至近 900 万名开发者，GitHub 仓库已获得 17.6 万个 Star 和 1.7 万个 Fork。 这笔巨额融资表明投资者对本地 AI 模型部署及更广泛的开源 AI 生态系统抱有强烈信心。Ollama 的大规模普及凸显了一个重要的行业趋势：越来越多的开发者出于隐私、成本效益和可用性的考虑，开始优先选择在自己的硬件上运行 AI 模型。 本轮融资由 Theory Ventures 领投，创始人兼首席执行官 Jeff Morgan 向 TechCrunch 证实了这一消息。Ollama 提供了命令行界面、本地 REST API、模型管理工具，以及将开放权重模型与编程助手和其他应用程序集成的功能。

rss · TechCrunch · 7月9日 13:00

**背景**: Ollama 是一个开源软件平台，旨在简化在本地计算机上运行和管理大型语言模型（LLM）的过程。它封装了设置本地 AI 环境所涉及的许多复杂性，使得下载并与 Llama、Qwen 或 DeepSeek 等模型进行交互变得像运行单个命令一样简单。该工具已成为本地 LLM 技术栈的核心组成部分，与 LM Studio 和 llama.cpp 等替代方案竞争。

## [开发者探索 Codeberg 和自托管 Gitea 等 GitHub 替代方案](https://www.howtogeek.com/why-developers-are-ditching-github-for-codeberg-and-self-hosting-alternatives/) ⭐️ 6.0/10

How-To Geek 的一篇文章强调了开发者从 GitHub 迁移到 Codeberg 和自托管解决方案的日益增长的趋势，主要出于对平台政策和控制权的担忧。然而，社区成员在很大程度上反驳了"大规模出走"的说法，指出实际只有极少数开发者进行了切换。 这一讨论反映了开发者社区中关于平台依赖的真实紧张关系，因为 GitHub 在微软旗下的主导地位引发了对不透明的审核政策、AI 训练使用代码以及供应商锁定的担忧。了解可用的替代方案有助于开发者对代码托管基础设施以及在便利性和自主权之间的权衡做出明智决策。 Codeberg 是一个非营利性、社区主导的平台，基于 Forgejo（Gitea 的一个分支）构建，专门为自由和开源项目提供免费的 Git 托管。Gitea 是一个用 Go 语言编写的轻量级自托管 Git 服务，采用 MIT 许可证，包含代码审查、通过 act runner 实现的 CI/CD、包注册表和团队协作功能。

hackernews · Gedxx · 7月9日 08:22 · [社区讨论](https://news.ycombinator.com/item?id=48842611)

**背景**: GitHub 于 2018 年被微软收购，托管了超过 1 亿开发者，已成为基于 Git 的代码托管和协作的事实标准。Codeberg 作为专注于 FOSS 项目的非营利替代方案而成立，运行在 Forgejo 上——Forgejo 本身是 2022 年因对 Gitea 企业治理方向的担忧而创建的社区治理分支。Gitea 最初于 2016 年从 Gogs 分支出来，由于其最低的资源要求和包括 CI/CD 和包管理在内的一体化功能集，已成为最受欢迎的自托管 Git 平台之一。
