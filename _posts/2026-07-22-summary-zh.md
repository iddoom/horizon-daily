## [Terrence Tao 使用 ChatGPT 探索 Jacobian 猜想反例](https://chatgpt.com/share/6a5fdc7a-d6f8-83e8-bbea-8deb42cfed56) ⭐️ 9.0/10

著名数学家 Terrence Tao 公开分享了一段 ChatGPT 对话记录，在其中他通过人机协作探索并简化了最近发现的 Jacobian 猜想反例。该反例在三维空间中被发现，推翻了这一长期存在的猜想在 N > 2 情况下的正确性。 这次互动展示了一种高效且新颖的人机协作模式：前沿 AI 作为顶级专家的"假设性"头脑风暴伙伴，加速了对新数学突破的消化和推广。它证明了尽管 AI 可能难以独立完成原创证明，但它在帮助领域专家将复杂的新知识映射到其现有的认知框架方面具有非凡的价值。 Jacobian 猜想提出，具有非零常数 Jacobian 行列式的多项式函数必定具有多项式逆函数，目前该猜想在二维（N=2）情况下仍是一个开放性问题。此次讨论的新反例适用于三维空间（N=3），Tao 利用 AI 迭代地提出简化方案，以更好地理解这一特定案例的底层机制。

hackernews · gmays · 7月22日 17:30 · [社区讨论](https://news.ycombinator.com/item?id=49010345)

**背景**: Jacobian 猜想是代数几何中一个著名的未解难题，涉及多变量多项式映射。Shreeram Abhyankar 曾广泛宣传这一问题，指出它是一个除了微积分基础知识外无需太多背景就能理解的难题，但它也因大量已发表和未发表的证明最终被发现含有微妙错误而臭名昭著。最近，数学家兼 Anthropic 员工 Levent Alpöge 展示了一个在三维空间中的明确反例，该反例是使用大语言模型发现的，从而推翻了该猜想在大于二维情况下的正确性。

一份新报告显示，OpenAI 计划到 2030 年在 AI 基础设施上投入约 7500 亿美元，这一惊人数字大致相当于瑞典的整个国内生产总值（GDP）。这是历史上单一公司最大的技术基础设施投资承诺之一。 这种前所未有的支出水平表明，AI 行业正在进入一场资本密集型的军备竞赛，竞争优势将越来越依赖于对大规模计算资源的获取。这样的投资将重塑全球硬件供应链，加速数据中心建设，并可能加剧 Google、Anthropic 和 Meta 等竞争对手追赶 OpenAI 基础设施雄心的压力。 7500 亿美元的数字涵盖了到 2030 年的基础设施支出，包括数据中心建设、GPU 和定制芯片采购、能源供应以及网络设备。这一承诺的规模引发了关于能源供应、半导体供应限制以及如此大规模资本支出的财务可持续性的重大疑问。

## [Bento：将完整演示文稿工具打包进单个 HTML 文件](https://bento.page/slides/) ⭐️ 7.0/10

一位开发者发布了开源演示文稿工具 Bento，它将编辑、展示、动画和实时协作功能全部打包进一个约 560 KB 的 HTML 文件中。该工具使用 reveal.js 和 Claude Code 构建，完全离线运行，无需云登录或安装，还可以直接将其提供给 Claude 或 ChatGPT，将现有的 PPTX 文件转换为 Bento 幻灯片。 Bento 解决了 AI 辅助开发中的一个实际工作流问题：此前使用编程工具构建幻灯片的团队即使要做小修改也得手动编辑代码。它还展示了本地优先软件的可行性，证明了功能完整且支持协作的应用程序可以作为便携、自包含的文件存在，而无需依赖远程服务器或云基础设施。 该 HTML 文件顶部包含一段纯 JSON 数据用于存储幻灯片内容——使其可被 AI 编程工具读取和检索——同时包含一个 base64 编码的应用程序块，通过 DecompressionStream 在浏览器中解压。实时协作使用加密的盲中继服务器，该服务器无法看到任何用户数据，整个代码库以 MIT 许可证在 GitHub 上开源。

hackernews · starfallg · 7月22日 15:19 · [社区讨论](https://news.ycombinator.com/item?id=49008211)

**背景**: 本地优先软件是一种工程方法，应用程序主要将数据存储在用户自己的设备上而非远程服务器，允许离线读写，并在网络可用时在后台同步数据。这一概念由 Ink & Switch 研究人员在 2019 年的论文中正式提出，强调用户对数据的所有权。单文件 HTML 应用程序代表了这一理念的极致形式，将所有代码、数据和依赖项打包到一个便携文件中，无需任何外部获取。加密盲中继是一种服务器架构，中继服务器在客户端之间传输端到端加密数据，但自身始终无法解密或读取内容。
